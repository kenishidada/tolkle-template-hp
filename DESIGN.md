# デザイン指針

AIがサイトを生成・修正する際に従うルール。

## タイポグラフィ
- フォント: System UI + Inter（globals.css で設定済み）
- サイズスケール: 14px / 16px / 20px / 24px / 32px / 48px
- 本文: 16px, line-height 1.7
- 見出し: font-weight 600-700

## レイアウト
- 最大幅: 1200px（max-w-7xl）
- 左右パディング: 24px（px-6）
- セクション間余白: 80px-120px（py-20 〜 py-30）

## カラー
- globals.css の CSS変数で定義済み
- 業種に合わせてアクセントカラーを変更する
- ニュートラルベース + 1アクセントカラーの組み合わせ
- コントラスト比 WCAG AA 準拠

## 余白
- 8の倍数: 8 / 12 / 16 / 24 / 32 / 48 / 64 / 80
- カード内: 24px（p-6）
- 要素間: 16px（gap-4）

## 角丸
- 大（セクション、カード）: 12px（rounded-lg）
- 中（ボタン、入力）: 10px（rounded-md）
- 小（バッジ、タグ）: 8px（rounded-sm）

## インタラクション
- ホバー: 薄い影 + 微小なtranslateY
- フォーカス: ring-2 ring-ring ring-offset-2
- トランジション: duration-200

## 画像
- アスペクト比を維持
- object-cover で切り抜き
- 角丸を親要素と統一

## レスポンシブ
- モバイルファースト
- ブレークポイント: sm(640) / md(768) / lg(1024)
- モバイルでは1カラム、タブレット以上で2-3カラム

## アイコン
- lucide-react を使用
- サイズ: 16px(小) / 20px(中) / 24px(大)

## やってはいけないこと
- グラデーション背景の多用
- 装飾的なSVGパターン
- 不自然に大きなpadding
- 全セクション同じレイアウトの繰り返し
- 過剰なアニメーション
